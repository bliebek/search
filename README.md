## First time run

In the project directory, you can run:

### `yarn && yarn start`

or 

### `npm install && npm run start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Day to day work

### `yarn start`

or

### `npm run start`


## Production build

### `yarn build`

or

### `npm run build`
